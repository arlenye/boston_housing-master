编程练习3：定义衡量标准¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

from sklearn.metrics import r2_score
def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    score = r2_score(y_true, y_predict)

    return score

问题 7- 网格搜索（Grid Search）¶

什么是网格搜索法？如何用它来优化模型？

问题 7 - 回答:¶

  1. 网格搜索法是指定参数的一种穷举搜索算法，通过将估值函数的参数通过交叉验证的方法进行优化来得到最优的学习算法。
  2. 网格搜索法是指将模型各个参数的可能取值排列组合，列出所有可能组合结果生成网格，然后将各个组合用于模型训练，并使用交叉验证对模型表现进行评估，在拟合算法尝试了所有的参数组合之后，返回一个带有最佳参数组合的模型，可以通过clf.bestparams 获得参数值。

问题 8 - 回答：¶

  1. k折交叉验证法默认把全部训练集按顺序分成k份，依次取k份中的1份作为测试集，剩下的k-1份作为训练集，进行k次训练和测试，得到k个结果，计算k次评估结果的平均值作为该模型的最终评估结果。k折交叉验证法设置shuffle =true可以随机打乱数据然后分成k份。
  2. 假如定义了一个kfold： kf = KFold(n_splits=2)，可以把kf作为参数传给GridSearchCV，这样他们就结合起来完成最佳参数组合的选择。例如：clf = GridSearchCV(estimator=svm, param_grid=p_grid, cv=kf)
  3. GridSearchCV中的'cvresults'属性可以告诉我们std_test_score,rank_test_score,各种计算花的时间，参数列表各组训练的分数等信息
  4. 网格搜索时如果不使用交叉验证会得不到最大限度的训练数据和测试数据，进而得不到最准确的精确度。 交叉验证充分利用了所有样本，得到的精确度更准确。

模型健壮性¶

一个最优的模型不一定是一个健壮模型。有的时候模型会过于复杂或者过于简单，以致于难以泛化新增添的数据；有的时候模型采用的学习算法并不适用于特定的数据结构；有的时候样本本身可能有太多噪点或样本过少，使得模型无法准确地预测目标变量。这些情况下我们会说模型是欠拟合的。
